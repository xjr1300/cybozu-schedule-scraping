# cybozu-schedule-scraping

## システム要件

- Python 3.11
- [Poetry](https://python-poetry.org/)

## インストール方法

```bash
git clone git@github.com:xjr1300/cybozu-schedule-scraping.git
cd cybozu-schedule-scraping
poetry install
```

## 使用方法

```bash
source .venv/bin/activate
python -m cybozu_schedule_scraping
```
